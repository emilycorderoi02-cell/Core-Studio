# CORE Studio — Portafolio Web

Sitio web de portafolio para **Emy**, diseñadora de productos y creadora de CORE Studio.

## 🚀 Cómo publicar en GitHub Pages

1. Sube esta carpeta completa a un repositorio en GitHub
2. Ve a **Settings → Pages**
3. En **Source**, selecciona `main` y carpeta `/ (root)`
4. Haz clic en **Save**
5. Tu sitio estará en: `https://TU-USUARIO.github.io/NOMBRE-REPOSITORIO`

## ✏️ Qué debes hacer

### 1. Agregar fotos
- En cada carpeta de proyecto pon la foto renombrada como **`foto.jpg`**
- En `LOGO CORE STUDIO/` pon la foto de Emy renombrada como **`foto.jpg`**

### 2. Editar descripciones
Abre cada `descripcion.txt` con el Bloc de notas y escribe el texto del proyecto.

### 3. Subir a GitHub
- Arrastra y suelta toda la carpeta en tu repositorio de GitHub
- Activa GitHub Pages en Settings → Pages
